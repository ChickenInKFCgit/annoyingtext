# Projet
## Description et Historique
Initialement créé le 5 octobre 2024, annoying text est un programme très basique permettant de transformer une phrase ou un texte en une phrase ou un texte
rédigé avec des caractères spéciaux, et ceux-ci sont piochés alétoirement parmis ceux ressemblant à la lettre correspondante.

Importé sur github le 22 juin 2026, ce projet a été revisité avec un stockage des données dans un fichier .tsv au lieu de l'initial dictionnaire des données
écrit en brut. De plus, ont été rajoutés des fonctions de chargement et de sauvegarde pour ce même fichier.
